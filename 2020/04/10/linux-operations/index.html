linux 操作

Posted on 2020-04-10 Edited on 2020-04-14

初步操作

  • echo 输入之后返回相同的内容
    • echo $HOME
  • pwd Print Working Directory 打印当前路径
  • cd Change directory
    • cd path / cd ~
    • cd ..
  • ls List Directory contents
    • ls
    • ls -l
    • ls -l -a = ls -la
    • ls -ln

寻求帮助

  • man Manual
  • -h 内部帮助
  • 面向StackOverFlow编程

文件内容

  • cat 打印文件内容 Concatenate and print files
    • cat abc.txt
      • cat abc.txt bbc.txt
      • cat < a.txt # read from stdin
  • head and tail 头尾
    • head a.txt
    • tail a.txt
    • tail -n 5 a.txt
    • tail -f a.txt #有更新内容就会打印出来
  • less 交互浏览
    • less a.txt #内容多的时候可能效果要好
  • / or grep 内容查找
    • man less | grep -n sim
    • cat 100.txt | grep -n word
    • /word
  • wc 单词统计 Word, line and byte count
    • man wc | wc -l

重定向和管道

  • 重定向：改变输入输出设备
    • 标准输入（stdin）/ 标准输出（stdout）：控制台/键盘/屏幕
    • echo hello > hello.txt
    • echo world >> hello.txt
    • cat < hello.txt
  • 管道：将前一个命令的标准输出作为下一个程序的标准输入
    • man less | grep sim
    • man less | grep -n sim | grep That > that.txt # multiple pipes
